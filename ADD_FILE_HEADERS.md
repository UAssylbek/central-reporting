### users.go (handler)
```go
/**
 * HTTP handlers для работы с пользователями
 *
 * Описание:
 * - CRUD операции с пользователями
 * - Пагинация и фильтрация
 * - Управление аватарами
 * - Audit logging всех операций
 *
 * Эндпоинты:
 * - GET /users - список пользователей (пагинация)
 * - GET /users/:id - получить пользователя
 * - POST /users - создать пользователя
 * - PUT /users/:id - обновить пользователя
 * - DELETE /users/:id - удалить пользователя
 * - POST /users/:id/avatar - загрузить аватар
 * - DELETE /users/:id/avatar - удалить аватар
 *
 * Права доступа:
 * - GET: admin, moderator
 * - POST: admin
 * - PUT: admin, moderator (ограничено), self
 * - DELETE: admin
 *
 * @package handlers
 */
```
